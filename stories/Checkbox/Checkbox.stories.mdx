## Props

| Property         | Type                                    | Description                         |
| ---------------- | --------------------------------------- | ----------------------------------- |
| `children`       | `ReactNode`                             | Checkbox label                      |
| `indeterminate`  | `boolean`                               | Show minus icon when checked        |
| `size`           | `null` \| `"small"` \| `"large"`        | Size                                |
| `loading`        | `boolean`                               | Show spinner                        |
| `loadingIcon`    | `ReactNode`                             | Custom spinner icon                 |
| `loadingDelay`   | `number`                                | Loading delay time, in milliseconds |
| `disabled`       | `boolean`                               | Disable change event                |
| `checked`        | `boolean`                               | Controlled checked                  |
| `defaultChecked` | `boolean`                               | Uncontrolled checked                |
| `onChange`       | `ChangeEventHandler<HTMLInputElement>`  | Change event                        |
| `inputRef`       | `Ref<HTMLInputElement>`                 | Ref to internal input element       |
| `inputProps`     | `InputHTMLAttributes<HTMLInputElement>` | Internal input props                |

It accepts all props which the native `<label>` element supports.
