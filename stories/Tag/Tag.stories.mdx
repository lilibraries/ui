### Clickable

When the `onClick` prop is specified, it is clickable by default. You can set `clickable` prop to `false` to disable it. Note that the `clickable` prop only effects component styles, not events. If you want to disable events, please use the `disabled` prop.

## Props

| Property     | Type                                                                             | Description                       |
| ------------ | -------------------------------------------------------------------------------- | --------------------------------- |
| `as`         | `ElementType`                                                                    | Element type, default is `"span"` |
| `children`   | `ReactNode`                                                                      | Tag content                       |
| `variant`    | `null` \| `"solid"` \| `"hollow"`                                                | Variant                           |
| `size`       | `null` \| `"small"` \| `"large"`                                                 | Size                              |
| `intent`     | `null` \| `"major"` \| `"minor"` \| `"positive"` \| `"alertive"` \| `"negative"` | Intent                            |
| `round`      | `boolean`                                                                        | Make corners round                |
| `borderless` | `boolean`                                                                        | Remove borders                    |
| `clickable`  | `boolean`                                                                        | Only effect styles                |
| `clearable`  | `boolean`                                                                        | Show clear icon                   |
| `clearIcon`  | `boolean`                                                                        | Custom clear icon                 |
| `disabled`   | `boolean`                                                                        | Disable clear and click evnets    |
| `onClear`    | `MouseEventHandler`                                                              | Clear event                       |
| `onClick`    | `MouseEventHandler`                                                              | Click event                       |

It accepts all props which the `as` prop supports.
