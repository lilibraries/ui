## API

| Property    | Type                                                                                                                                                                                       | Description                                                                                                       |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------- |
| `size`      | `null` \| `"small"` \| `"large"`                                                                                                                                                           | Size.                                                                                                             |
| `variant`   | `null` \| `"solid"` \| `"dotted"`                                                                                                                                                          | Variant.                                                                                                          |
| `color`     | `null` \| `"red"` \| `"magenta"` \| `"purple"` \| `"indigo"` \| `"navy"` \| `"blue"` \| `"cyan"` \| `"teal"` \| `"green"` \| `"lime"` \| `"yellow"` \| `"orange"` \| `"brown"` \| `"gray"` | Color.                                                                                                            |
| `rounded`   | `boolean`                                                                                                                                                                                  | Make tag corners round. Only works when the `variant` is not `"dotted"`.                                          |
| `animated`  | `boolen`                                                                                                                                                                                   | Perform heartbeat animation. Only works when the `variant` is `"dotted"`.                                         |
| `outlined`  | `boolen`                                                                                                                                                                                   | Add outline to the tag.                                                                                           |
| `tag`       | `ReactNode`                                                                                                                                                                                | Custom badge tag.                                                                                                 |
| `count`     | `number`                                                                                                                                                                                   | Count number.                                                                                                     |
| `maxCount`  | `number`                                                                                                                                                                                   | If the `count` is large than `maxCount`, the tag will show `+` symbol.                                            |
| `showZero`  | `boolean`                                                                                                                                                                                  | The tag will hide when the `count` is zero by default. This property is used to modify the default behavior.      |
| `invisible` | `boolean`                                                                                                                                                                                  | Control the tag visibility.                                                                                       |
| `placement` | `"top-start"` \| `"top-end"` \| `"bottom-start"` \| `"bottom-end"`                                                                                                                         | Specify the tag placement. Default is `"top-end"`.                                                                |
| `offset`    | `number` \| `string` \| [`number` \| `string`, `number` \| `string`]                                                                                                                       | Placement offset. When it is an array, the first value is horizontal offset, the second value is vertical offset. |

It accepts all props which the native `<span>` element supports.
