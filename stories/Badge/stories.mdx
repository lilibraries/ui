## API

| Property    | Type                                                                                                                                                                                       | Description                                                                                                  |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------ |
| `size`      | `null` \| `"small"` \| `"large"`                                                                                                                                                           | Size value.                                                                                                  |
| `variant`   | `null` \| `"solid"` \| `"dotted"`                                                                                                                                                          | Variant value.                                                                                               |
| `color`     | `null` \| `"red"` \| `"magenta"` \| `"purple"` \| `"indigo"` \| `"navy"` \| `"blue"` \| `"cyan"` \| `"teal"` \| `"green"` \| `"lime"` \| `"yellow"` \| `"orange"` \| `"brown"` \| `"gray"` | Color value.                                                                                                 |
| `rounded`   | `boolean`                                                                                                                                                                                  | Make tag corners round. Works when the `variant` is not `"dotted"`.                                          |
| `animated`  | `boolen`                                                                                                                                                                                   | Perform heartbeat animation. Only works when the `variant` is `"dotted"`.                                    |
| `outlined`  | `boolen`                                                                                                                                                                                   | Add outline to the tag.                                                                                      |
| `tag`       | `ReactNode`                                                                                                                                                                                | Custom badge tag.                                                                                            |
| `count`     | `number`                                                                                                                                                                                   | Count number.                                                                                                |
| `maximum`   | `number`                                                                                                                                                                                   | If the `count` is large than `maximum`, the tag will show `+` symbol.                                        |
| `zeroable`  | `boolean`                                                                                                                                                                                  | The tag will hide when the `count` is zero by default. This property is used to change the default behavior. |
| `invisible` | `boolean`                                                                                                                                                                                  | Control the tag visibility.                                                                                  |
| `offset`    | `number` \| `string` \| <code>{ x?: string \| number; y?: string \| number }</code>                                                                                                        | When it is an object, the `x` is horizontal offset, the `y` is vertical offset.                              |
| `placement` | `"top-start"` \| `"top-end"` \| `"bottom-start"` \| `"bottom-end"`                                                                                                                         | Specify the tag placement. Default is `"top-end"`.                                                           |

It accepts all props which the native `<span>` element supports.
