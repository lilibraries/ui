## Props

| Property          | Type                                                                                                 | Description                            |
| ----------------- | ---------------------------------------------------------------------------------------------------- | -------------------------------------- |
| `vertical`        | `boolean`                                                                                            | Vertical display                       |
| `variant`         | `null` &#124; `"solid"` &#124; `"outlined"` &#124; `"flat"`                                          | Variant                                |
| `intent`          | `null` &#124; `"major"` &#124; `"minor"` &#124; `"positive"` &#124; `"alertive"` &#124; `"negative"` | Intent                                 |
| `size`            | `null` &#124; `"small"` &#124; `"large"`                                                             | Size                                   |
| `fluid`           | `boolean`                                                                                            | Display like a block element           |
| `round`           | `boolean`                                                                                            | Make corners round                     |
| `truncated`       | `boolean`                                                                                            | Truncate text when it exceeds one line |
| `disabled`        | `boolean`                                                                                            | Disabled state                         |
| `iconOnly`        | `boolean`                                                                                            | Show icon only                         |
| `loading`         | `boolean`                                                                                            | Show spinner                           |
| `loadingPosition` | `null` &#124; `"start"` &#124; `"center"` &#124; `"end"`                                             | Specify spinner position               |

It accepts all props which the native `<div>` element supports.
