## Props

| Property   | Type                                                                                                                                                                                                                                                               | Description                 |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------- |
| `size`     | `null` &#124; `"small"` &#124; `"large"`                                                                                                                                                                                                                           | Size                        |
| `intent`   | `null` &#124; `"major"` &#124; `"minor"` &#124; `"positive"` &#124; `"alertive"` &#124; `"negative"`                                                                                                                                                               | Intent                      |
| `color`    | `null` &#124; `"red"` &#124; `"magenta"` &#124; `"purple"` &#124; `"indigo"` &#124; `"navy"` &#124; `"blue"` &#124; `"cyan"` &#124; `"teal"` &#124; `"green"` &#124; `"lime"` &#124; `"yellow"` &#124; `"orange"` &#124; `"brown"` &#124; `"gray"` &#124; `string` | Color                       |
| `animated` | `boolean`                                                                                                                                                                                                                                                          | Perform heartbeat animation |

It accepts all props which the native `<span>` element supports.
