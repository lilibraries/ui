## Props

| Property   | Type                                                                                                                                                                                         | Description                 |
| ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------- |
| `size`     | `null` \| `"small"` \| `"large"`                                                                                                                                                             | Size                        |
| `color`    | `"red"` \| `"magenta"` \| `"purple"` \| `"indigo"` \| `"navy"` \| `"blue"` \| `"cyan"` \| `"teal"` \| `"green"` \| `"lime"` \| `"yellow"` \| `"orange"` \| `"brown"` \| `"gray"` \| `string` | Color                       |
| `animated` | `boolean`                                                                                                                                                                                    | Perform heartbeat animation |

It accepts all props which the native `<span>` element supports.
