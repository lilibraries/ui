## API

| Property   | Type                                                                                                                                                                                                   | Description       |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------- |
| `size`     | `null` \| `"small"` \| `"large"`                                                                                                                                                                       | Size              |
| `color`    | `null` \| `"red"` \| `"magenta"` \| `"purple"` \| `"indigo"` \| `"navy"` \| `"blue"` \| `"cyan"` \| `"teal"` \| `"green"` \| `"lime"` \| `"yellow"` \| `"orange"` \| `"brown"` \| `"gray"` \| `string` | Color             |
| `animated` | `boolean`                                                                                                                                                                                              | Perform animation |

It accepts all props which the native `<span>` element supports, except the `children` prop. This component does not have `children` prop.
