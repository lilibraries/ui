## API

### Avatar

| Property   | Type                             | Description                        |
| ---------- | -------------------------------- | ---------------------------------- |
| `as`       | `ElementType`                    | Element type, default is `"span"`. |
| `size`     | `null` \| `"small"` \| `"large"` | Avatar size.                       |
| `image`    | `string`                         | Image url.                         |
| `rounded`  | `boolean`                        | Make corners rounded.              |
| `outlined` | `boolean`                        | Add outline.                       |

It accepts all props which the `as` element type supports.

### Avatar.Group

| Property   | Type                             | Description           |
| ---------- | -------------------------------- | --------------------- |
| `size`     | `null` \| `"small"` \| `"large"` | Avatar size.          |
| `rounded`  | `boolean`                        | Make corners rounded. |
| `outlined` | `boolean`                        | Add outline.          |

It accepts all props which the native `<div>` element supports.
