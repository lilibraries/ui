# Direction

Configure document or element direction.

## Examples

### Global

The `Direction` button in the document toolbar is used to set the `<html>`'s `dir` attribute, you can click it to try.

```tsx
import React from "react";
import { Direction } from "@lilib/ui";

function App() {
  return <Direction value="rtl">...</Direction>;
}
```

### Scoped

If the `scoped` prop is specified to `true`, the `children` prop must be a React element that accepts `dir` prop.

## API

### Direction

| Property   | Type                          | Description                                |
| ---------- | ----------------------------- | ------------------------------------------ |
| `value`    | `"ltr"` \| `"rtl"`            | **Required** direction value.              |
| `scoped`   | `boolean`                     | Scoped.                                    |
| `children` | `ReactElement` \| `ReactNode` | **Required** child element or child nodes. |

### Direction.useConfig

```ts
function useConfig(): "ltr" | "rtl";
```

Get configured direction.

### Direction.Context

A context created by `React.createContext`. It's default value is `"ltr"`.
