# Duration

Duration component can configure animation durations. It has two functions: one is to configure CSS animation (including transition) durations, and the other is to configure JavaScript animation durations.

## Examples

### Global

Setting durations globally will add styles to `<html>`.

```tsx
import React from "react";
import { Duration } from "@lilib/ui";

function App() {
  return (
    <Duration base={400} fast={200} slow={800} lazy={1200}>
      ...
    </Duration>
  );
}
```

### Scoped

Many components in this library perform animation effects. The following example describes how to configure scoped durations. The Spinner component performs a rotation animation of 750 milliseconds by default, but 3000 milliseconds in this example. Scoped way will add styles to the child element, and the `children` prop must be a React element that accepts `style` prop.

## API

### Duration

| Property   | Type                          | Description                               | Default |
| ---------- | ----------------------------- | ----------------------------------------- | ------- |
| `children` | `ReactElement` \| `ReactNode` | **Required** child element or child nodes | -       |
| `scoped`   | `boolean`                     | Scoped                                    | -       |
| `base`     | `number`                      | Base animation duration, in milliseconds  | `300`   |
| `fast`     | `number`                      | Fast animation duration, in milliseconds  | `150`   |
| `slow`     | `number`                      | Slow animation duration, in milliseconds  | `750`   |
| `lazy`     | `number`                      | Lazy animation duration, in milliseconds  | `1250`  |

Duration will inherit the upper configuration.

### Duration.useConfig

```ts
function useConfig(): {
  base: number;
  fast: number;
  slow: number;
  lazy: number;
};
```

Get configured durations.

### Duration.Context

A context created by `React.createContext`. It's default value is `{ base: 300, fast: 150, slow: 750, lazy: 1250 }`.
