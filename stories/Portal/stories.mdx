## API

## Portal

| Property    | Type                                                                                        | Description                                                                                                                                                                                           |
| ----------- | ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `children`  | `ReactNode`                                                                                 | The content to be rendered into the `container`                                                                                                                                                       |
| `container` | [`EffectTarget<HTMLElement>`](https://lilibraries.github.io/utils/types#effecttarget-react) | The DOM node where the content should be rendered. It can be `null`, `undefined`, a DOM element, a function returns a DOM element or a ref object refrences a DOM element. Default is `document.body` |

### Portal.Config

| Property    | Type                                                                                        | Description                                                                                                                                                                       |
| ----------- | ------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `children`  | `ReactNode`                                                                                 | **Required** subcomponents                                                                                                                                                        |
| `container` | [`EffectTarget<HTMLElement>`](https://lilibraries.github.io/utils/types#effecttarget-react) | The DOM node where the portal content should be rendered. It can be `null`, `undefined`, a DOM element, a function returns a DOM element or a ref object refrences a DOM element. |

Portal.Config will inherit the upper configuration.

### Portal.Config.useConfig

```ts
function useConfig(): { container?: EffectTarget<HTMLElement> };
```

Get configuration.

### Portal.Config.Context

A context created by `React.createContext`. It's default value is an empty object `{}`.
