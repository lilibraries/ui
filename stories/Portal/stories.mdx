This component renders children into a DOM node by using `ReactDOM.createPortal`.

## API

| Property    | Type                                                                                        | Description                                     |
| ----------- | ------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| `children`  | `ReactNode`                                                                                 | The content to be rendered into the `container` |
| `container` | [`EffectTarget<HTMLElement>`](https://lilibraries.github.io/utils/types#effecttarget-react) | A HTML element, default is `document.body`      |

The `container` prop can be `null`, `undefined`, a DOM element, a function returns a DOM element or a ref object refrences a DOM element.
