# Portal

This component renders children into a DOM node by using `ReactDOM.createPortal`.

## Props

| Property    | Type                         | Description                                     |
| ----------- | ---------------------------- | ----------------------------------------------- |
| `children`  | `ReactNode`                  | The content to be rendered into the `container` |
| `container` | `TargetCreator<HTMLElement>` | A HTML element, default is `document.body`      |

```ts
type Target<T> = T | undefined | null;
type TargetCreator<T> =
  | Target<T>
  | (() => Target<T>)
  | React.RefObject<Target<T>>
  | React.MutableRefObject<Target<T>>;
```
