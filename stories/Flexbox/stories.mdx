## API

| Property    | Type                                                                                                      | Description                                                                                                                                      |
| ----------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `fluid`     | `boolean`                                                                                                 | Display as a block element                                                                                                                       |
| `direction` | `"row"` \| `"column"` \| `"row-reverse"` \| `"column-reverse"`                                            | [`flex-direction`](https://developer.mozilla.org/en-US/docs/Web/CSS/flex-direction)                                                              |
| `gap`       | `"1x"` \| `"2x"` \| `"3x"` \| `"4x"` \| `"5x"` \| `"6x"` \| `"7x"` \| `"8x"` \| `"9x"`                    | Gap size                                                                                                                                         |
| `wrap`      | `true` \| `false` \| `"nowrap"` \| `"wrap"` \| `"wrap-reverse"`                                           | [`flex-wrap`](https://developer.mozilla.org/en-US/docs/Web/CSS/flex-wrap). `true` is equivalent to `"wrap"`; `false` is equivalent to `"nowrap"` |
| `align`     | `"stretch"` \| `"center"` \| `"baseline"` \| `"flex-start"` \| `"flex-end"`                               | [`align-items`](https://developer.mozilla.org/en-US/docs/Web/CSS/align-items)                                                                    |
| `justify`   | `"flex-start"` \| `"flex-end"` \| `"center"` \| `"space-between"` \| `"space-around"` \| `"space-evenly"` | [`justify-content`](https://developer.mozilla.org/en-US/docs/Web/CSS/justify-content)                                                            |

It accepts all props which the native `<div>` element supports.
