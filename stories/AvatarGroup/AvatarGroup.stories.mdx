## Props

| Property    | Type                                                                                                                                                                                       | Description                                         |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------- |
| `size`      | `null` \| `"small"` \| `"large"`                                                                                                                                                           | Size                                                |
| `round`     | `boolean`                                                                                                                                                                                  | Make corners round                                  |
| `color`     | `null` \| `"red"` \| `"magenta"` \| `"purple"` \| `"indigo"` \| `"navy"` \| `"blue"` \| `"cyan"` \| `"teal"` \| `"green"` \| `"lime"` \| `"yellow"` \| `"orange"` \| `"brown"` \| `"gray"` | Color                                               |
| `outlined`  | `boolean`                                                                                                                                                                                  | Add outline                                         |
| `clickable` | `boolean`                                                                                                                                                                                  | Show hover and active styles. It only effect styles |

It accepts all props which the native `<div>` element supports.
