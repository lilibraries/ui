# Baseline

This component resets browser default styles when when using on-demand import tools.

## Usage

It is highly recommended that you use this component in the entry component only once.

```tsx
import React from "react";
import { Baseline } from "@lilib/ui";

function App() {
  return <Baseline />;
}
// Or
function App() {
  return <Baseline>...</Baseline>;
}
```

You can also use the following ways instead.

```ts
import "@lilib/styles/dist/css/Baseline/index.css";
// Or
import "@lilib/styles/dist/scss/Baseline/index.scss";
```
