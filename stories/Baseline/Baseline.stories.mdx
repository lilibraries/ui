# Baseline

This component is a placeholder component to import global styles to reset default styles of some elements. It is recommended to use it only once in the application.

## Usage

```tsx
import React from "react";
import { Baseline } from "@lilib/ui";

function App() {
  return <Baseline />;
}
// Or
function App() {
  return <Baseline>...</Baseline>;
}
```
