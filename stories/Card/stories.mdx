## API

| Property         | Type                                          | Description                                                                                                                                                                             |
| ---------------- | --------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `children`       | `ReactNode`                                   | Card Content.                                                                                                                                                                           |
| `icon`           | `ReactNode`                                   | Header icon.                                                                                                                                                                            |
| `title`          | `ReactNode`                                   | Header title.                                                                                                                                                                           |
| `headnote`       | `ReactNode`                                   | Header end note.                                                                                                                                                                        |
| `headmark`       | `ReactNode`                                   | Header end mark.                                                                                                                                                                        |
| `footnote`       | `ReactNode`                                   | Footer start note.                                                                                                                                                                      |
| `footmark`       | `ReactNode`                                   | Footer end mark.                                                                                                                                                                        |
| `image`          | `string` \| `ReactNode`                       | Image url or other React node.                                                                                                                                                          |
| `imageSize`      | `string` \| `number`                          | Image size, default is `"100px"`. It set the image height when the `imagePlacement` is `"top"` or `"bottom"`. It set the image width when the `imagePlacement` is `"start"` or `"end"`. |
| `imagePlacement` | `"top"` \| `"bottom"` \| `"start"` \| `"end"` | Image placement, default is `"top"`.                                                                                                                                                    |
| `divided`        | `boolean`                                     | Divided header and footer by border.                                                                                                                                                    |
| `striped`        | `boolean`                                     | Show background color of header and footer.                                                                                                                                             |
| `shadowed`       | `boolean`                                     | Show shadow.                                                                                                                                                                            |
| `hoverable`      | `boolean`                                     | Show hover shadow and active shadow.                                                                                                                                                    |
| `unpadding`      | `boolean`                                     | Remove content padding.                                                                                                                                                                 |
| `borderless`     | `boolean`                                     | Remove borders.                                                                                                                                                                         |

It accepts all props which the `as` element type supports.
