## API

| Property   | Type                                                                                                           | Description                                  |
| ---------- | -------------------------------------------------------------------------------------------------------------- | -------------------------------------------- |
| `as`       | `ElementType`                                                                                                  | Element type, default is `"hr"`              |
| `vertical` | `boolean`                                                                                                      | Display vertical Line                        |
| `inline`   | `boolean`                                                                                                      | The `width` or `height` equals the font size |
| `spacing`  | `"1x"` \| `"2x"` \| `"3x"` \| `"4x"` \| `"5x"` \| `"6x"` \| `"7x"` \| `"8x"` \| `"9x"` \| `string` \| `number` | Spacing with other elements                  |

It accepts all props which the `as` element type supports.
