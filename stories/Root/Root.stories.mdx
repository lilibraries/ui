# Root

The only purpose of this component is to provide a placeholder to import and add CSS variables to the `<html>` element. It is recommended to use it only once in the application.

## Example

```tsx
import React from "react";
import { Root } from "@lilib/ui";

function App() {
  return <Root />;
}
// Or
function App() {
  return <Root>...</Root>;
}
```
