# Root

The only purpose of this component is to add CSS variables for the `<html>` element when importing on demand. It is recommended to use it only once in the application.

## Usage

```tsx
import React from "react";
import { Root } from "@lilib/ui";

function App() {
  return <Root />;
}
// Or
function App() {
  return <Root>...</Root>;
}
```

## API

| Property   | Type        | Description   |
| ---------- | ----------- | ------------- |
| `children` | `ReactNode` | Subcomponents |
