## Props

| Property    | Type                                                                             | Description              |
| ----------- | -------------------------------------------------------------------------------- | ------------------------ |
| `children`  | `ReactNode`                                                                      | Content                  |
| `intent`    | `null` \| `"major"` \| `"minor"` \| `"positive"` \| `"alertive"` \| `"negative"` | Intent                   |
| `icon`      | `ReactNode`                                                                      | Icon                     |
| `open`      | `boolean`                                                                        | Controlled open          |
| `closable`  | `boolean`                                                                        | Whether it can be closed |
| `closeIcon` | `boolean`                                                                        | Close icon               |
| `onClose`   | `(event: MouseEvent<HTMLButtonElement>) => void`                                 | Close event              |

It accepts all props which the native `<div>` element supports.
