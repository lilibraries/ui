# Prefix

Prefix is used to configure the CSS class name prefix and CSS variable name prefix of subcomponents.

## Example

```tsx
import React from "react";
import { Prefix, Button } from "@lilib/ui";

function App() {
  return (
    <Prefix cls="hello-" var="hello-">
      <Button /> {/* => "hello-button" */}
      <Subcomponent />
    </Prefix>
  );
}

function Subcomponent() {
  const {
    cls, // => "hello-"
    var, // => "hello-"
  } = Prefix.useConfig();
  ...
}
```

## API

### Prefix

| Property   | Type        | Description                           |
| ---------- | ----------- | ------------------------------------- |
| `cls`      | `string`    | **Required** CSS class name prefix    |
| `var`      | `string`    | **Required** CSS variable name prefix |
| `children` | `ReactNode` | **Required** subcomponents            |

### Prefix.useConfig

```ts
function useConfig(): { cls: string; var: string };
```

Get configured CSS class name prefix and CSS variable name prefix.

### Prefix.Context

A context created by `React.createContext`. It's default value is `{ cls: "li-", var: "li-" }`.
