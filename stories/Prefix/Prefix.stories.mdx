# Prefix

This component is used to set the CSS class name prefix and CSS variable name prefix of subcomponents. It is recommended to set it only once in the root component.

## Example

```tsx
import React from "react";
import { Prefix, Button } from "@lilib/ui";

function Root() {
  return (
    <Prefix cls="hello-" var="hello-">
      <Button /> {/* => "hello-button" */}
      <Subcomponent />
    </Prefix>
  );
}

function Subcomponent() {
  const {
    cls, // => "hello-"
    var, // => "hello-"
  } = Prefix.useConfig();
  ...
}
```

## API

### Prefix

| Property   | Type        | Description              | Default |
| ---------- | ----------- | ------------------------ | ------- |
| `cls`      | `string`    | CSS class name prefix    | "li-"   |
| `var`      | `string`    | CSS variable name prefix | "li-"   |
| `children` | `ReactNode` | Subcomponents            | -       |

### Prefix.useConfig()

```ts
function useConfig(): { cls: string; var: string };
```

Get configured CSS class name prefix.

### Prefix.Context

A context created by `React.createContext`. It's default value is `{ cls: "li-", var: "li-" }`.
