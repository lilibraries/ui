Image component extends the native `<img>` element, and supports error fallback and image placeholder, and improves lazy loading that some browsers don't support.

## API

| Property      | Type     | Description                  |
| ------------- | -------- | ---------------------------- |
| `fallback`    | `string` | Fallback image source url    |
| `placeholder` | `string` | Placeholder image source url |

It accepts all props which the native [`<img>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/img) element supports.
