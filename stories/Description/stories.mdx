## API

### Description

| Property   | Type        | Description         |
| ---------- | ----------- | ------------------- |
| `icon`     | `ReactNode` | Icon.               |
| `mark`     | `ReactNode` | End mark.           |
| `title`    | `ReactNode` | Description title.  |
| `children` | `ReactNode` | Description detail. |

It accepts all props which the native `<div>` element supports.
