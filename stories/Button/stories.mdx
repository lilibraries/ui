## Props

### Button

| Property           | Type                                                                             | Description                         |
| ------------------ | -------------------------------------------------------------------------------- | ----------------------------------- |
| `as`               | `ElementType`                                                                    | Element type, default is `"button"` |
| `children`         | `ReactNode`                                                                      | Button content                      |
| `size`             | `null` \| `"small"` \| `"large"`                                                 | Size                                |
| `variant`          | `null` \| `"solid"` \| `"hollow"`                                                | Variant                             |
| `intent`           | `null` \| `"major"` \| `"minor"` \| `"positive"` \| `"alertive"` \| `"negative"` | Intent                              |
| `fluid`            | `boolean`                                                                        | Display as a block element          |
| `round`            | `boolean`                                                                        | Make corners round                  |
| `truncated`        | `boolean`                                                                        | Truncate text                       |
| `borderless`       | `boolean`                                                                        | Remove borders                      |
| `active`           | `boolean`                                                                        | Active state                        |
| `disabled`         | `boolean`                                                                        | Disabled state                      |
| `startIcon`        | `ReactNode`                                                                      | Start icon                          |
| `endIcon`          | `ReactNode`                                                                      | End icon                            |
| `iconOnly`         | `boolean`                                                                        | Show icon only                      |
| `loading`          | `boolean`                                                                        | Show spinner                        |
| `loadingIcon`      | `ReactNode`                                                                      | Custom spinner icon                 |
| `loadingDelay`     | `number`                                                                         | Loading delay time, in milliseconds |
| `loadingPlacement` | `null` \| `"start"` \| `"center"` \| `"end"`                                     | Specify spinner placement           |

It accepts all props which the `as` element type supports.
