## API

### Button

| Property           | Type                                                                             | Description                                     |
| ------------------ | -------------------------------------------------------------------------------- | ----------------------------------------------- |
| `as`               | `ElementType`                                                                    | Element type, default is `"button"`.            |
| `children`         | `ReactNode`                                                                      | Button content.                                 |
| `size`             | `null` \| `"small"` \| `"large"`                                                 | Size.                                           |
| `intent`           | `null` \| `"major"` \| `"minor"` \| `"positive"` \| `"alertive"` \| `"negative"` | Intent.                                         |
| `variant`          | `null` \| `"solid"` \| `"hollow"`                                                | Variant.                                        |
| `fluid`            | `boolean`                                                                        | Display as a block element.                     |
| `rounded`          | `boolean`                                                                        | Make corners rounded.                           |
| `truncated`        | `boolean`                                                                        | Truncate long text.                             |
| `borderless`       | `boolean`                                                                        | Remove borders.                                 |
| `icon`             | `ReactNode`                                                                      | Start icon.                                     |
| `endIcon`          | `ReactNode`                                                                      | End icon.                                       |
| `iconOnly`         | `boolean`                                                                        | Show icon only.                                 |
| `active`           | `boolean`                                                                        | Active.                                         |
| `disabled`         | `boolean`                                                                        | Disabled.                                       |
| `loading`          | `boolean`                                                                        | Show spinner.                                   |
| `loadingIcon`      | `ReactNode`                                                                      | Custom spinner icon.                            |
| `loadingDelay`     | `number`                                                                         | Loading delay time, in milliseconds.            |
| `loadingPlacement` | `"auto"` \| `"start"` \| `"center"` \| `"end"`                                   | Specify spinner placement, default is `"auto"`. |

It accepts all props which the `as` element type supports. The `loadingIcon` and `loadingDelay` props can be configured by the [`Spinner.Config`](?path=/story/feedbacks-spinner--page#spinnerconfig) component.

### Button.Group

| Property           | Type                                                                             | Description                                     |
| ------------------ | -------------------------------------------------------------------------------- | ----------------------------------------------- |
| `size`             | `null` \| `"small"` \| `"large"`                                                 | Size.                                           |
| `intent`           | `null` \| `"major"` \| `"minor"` \| `"positive"` \| `"alertive"` \| `"negative"` | Intent.                                         |
| `variant`          | `null` \| `"solid"` \| `"hollow"`                                                | Variant.                                        |
| `vertical`         | `boolean`                                                                        | Vertical display.                               |
| `fluid`            | `boolean`                                                                        | Display as a block element.                     |
| `rounded`          | `boolean`                                                                        | Make corners rounded.                           |
| `truncated`        | `boolean`                                                                        | Truncate text.                                  |
| `borderless`       | `boolean`                                                                        | Remove borders.                                 |
| `iconOnly`         | `boolean`                                                                        | Show icon only.                                 |
| `disabled`         | `boolean`                                                                        | Disabled.                                       |
| `loading`          | `boolean`                                                                        | Show spinner.                                   |
| `loadingIcon`      | `ReactNode`                                                                      | Custom spinner icon.                            |
| `loadingDelay`     | `number`                                                                         | Loading delay time, in milliseconds.            |
| `loadingPlacement` | `auto` \| `"start"` \| `"center"` \| `"end"`                                     | Specify spinner placement, default is `"auto"`. |

It accepts all props which the native `<div>` element supports.
