# Transition

This component is a combination of the Transition and CSSTransition components in [react-transition-group](https://reactcommunity.org/react-transition-group/). The biggest difference is that this component can delay entering and exiting.

### CSS Transition

You can set `classes` prop to enable CSS transition. `classes` has three types:

- `boolean`: Use built-in class names, such as `li-transition-enter`.
- `string`: Prefix the class names with a string.
- `object`: Set the class names of each state separately.

The Transition component will automatically add the corresponding class name for the child element according to different states.

## API

### Transition

| Property      | Type                                                          | Description                                                                                                    |
| ------------- | ------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| `children`    | `ReactElement` \| `(state: TransitionState) => ReactElement)` | **Required.** Must be a React element, or a function returns a React element.                                  |
| `durations`   | `number` \| `TransitionDurations`                             | **Required.** Specify transition durations. It is usually consistent with the duration time of CSS transition. |
| `classes`     | `boolean` \| `string` \| `TransitionClasses`                  | Specify class names in different states.                                                                       |
| `in`          | `boolean`                                                     | Enter when `true`, exit when `false`.                                                                          |
| `enterDelay`  | `number`                                                      | Delay time on enter, in milliseconds.                                                                          |
| `exitDelay`   | `number`                                                      | Delay time on exit, in milliseconds.                                                                           |
| `firstMount`  | `boolean`                                                     | Mount on first time, no matter whether the component is entered.                                               |
| `firstAppear` | `boolean`                                                     | Perform enter phase when the `in` prop is initially true.                                                      |
| `keepMounted` | `boolean`                                                     | Keep mounted after the component is exited.                                                                    |
| `onEnter`     | `() => void`                                                  | State changes to `"enter"`.                                                                                    |
| `onEntering`  | `() => void`                                                  | State changes to `"entering"`.                                                                                 |
| `onEntered`   | `() => void`                                                  | State changes to `"entered"`.                                                                                  |
| `onExit`      | `() => void`                                                  | State changes to `"exit"`.                                                                                     |
| `onExiting`   | `() => void`                                                  | State changes to `"exiting"`.                                                                                  |
| `onExited`    | `() => void`                                                  | State changes to `"exited"`.                                                                                   |

```ts
type TransitionState = "enter" | "entering" | "entered" | "exit" | "exiting" | "exited";

interface TransitionDurations {
  entering: number;
  exiting: number;
}

interface TransitionClasses {
  enter?: string | boolean;
  entering?: string | boolean;
  entered?: string | boolean;
  exit?: string | boolean;
  exiting?: string | boolean;
  exited?: string | boolean;
}
```

### Static Properties

```ts
Transition.ENTER = "enter";
Transition.ENTERING = "entering";
Transition.ENTERED = "entered";
Transition.EXIT = "exit";
Transition.EXITING = "exiting";
Transition.EXITED = "exited";
```
