## API

| Property    | Type                                                                             | Description                        |
| ----------- | -------------------------------------------------------------------------------- | ---------------------------------- |
| `as`        | `ElementType`                                                                    | Element type, default is `"span"`. |
| `size`      | `null` \| `"small"` \| `"smaller"` \| `"large"` \| `"larger"`                    | Size value.                        |
| `intent`    | `null` \| `"major"` \| `"minor"` \| `"positive"` \| `"alertive"` \| `"negative"` | Intent value.                      |
| `muted`     | `boolean`                                                                        | Muted status.                      |
| `active`    | `boolean`                                                                        | Active status.                     |
| `disabled`  | `boolean`                                                                        | Disabled status.                   |
| `hoverable` | `boolean`                                                                        | Hoverable status.                  |

It accepts all props which the `as` element supports.
