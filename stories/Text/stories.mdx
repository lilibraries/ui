## API

| Property    | Type                                                                                                                                                                                       | Description                       |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------- |
| `as`        | `ElementType`                                                                                                                                                                              | Element type, default is `"span"` |
| `intent`    | `null` \| `"major"` \| `"minor"` \| `"positive"` \| `"alertive"` \| `"negative"`                                                                                                           | Intent                            |
| `color`     | `null` \| `"red"` \| `"magenta"` \| `"purple"` \| `"indigo"` \| `"navy"` \| `"blue"` \| `"cyan"` \| `"teal"` \| `"green"` \| `"lime"` \| `"yellow"` \| `"orange"` \| `"brown"` \| `"gray"` | Color                             |
| `muted`     | `boolean`                                                                                                                                                                                  | Muted                             |
| `active`    | `boolean`                                                                                                                                                                                  | Active                            |
| `disabled`  | `boolean`                                                                                                                                                                                  | Disabled                          |
| `hoverable` | `boolean`                                                                                                                                                                                  | Hoverable                         |

It accepts all props which the `as` element type supports.
