## API

### Size

| Property   | Type                             | Description                |
| ---------- | -------------------------------- | -------------------------- |
| `children` | `ReactNode`                      | **Required** subcomponents |
| `value`    | `null` \| `"small"` \| `"large"` | **Required** size value    |

### Size.useConfig

```ts
function useConfig(): null | "small" | "large";
```

Get configured size value.

### Size.Context

A context created by `React.createContext`. It's default value is `null`.
