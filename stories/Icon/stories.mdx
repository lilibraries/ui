# Icon

This component is just a wrapper for correcting icon alignment with text content around. This project doesn't provide a collection of icons. You can find many icon libraries in the open source community. [react-icons](https://react-icons.github.io/react-icons) is recommended, and it combines many other free icon libraries.

## API

| Property   | Type        | Description                                   |
| ---------- | ----------- | --------------------------------------------- |
| `icon`     | `ReactNode` | Element wrapped by Icon component.            |
| `spinning` | `boolean`   | Whether to perform rotate animation.          |
| `children` | `ReactNode` | Same as `icon` prop, but with lower priority. |

It accepts all props which the native `<span>` element supports.
