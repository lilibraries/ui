## API

### Intent

| Property   | Type                                                                                                 | Description   | Default |
| ---------- | ---------------------------------------------------------------------------------------------------- | ------------- | ------- |
| `value`    | `null` &#124; `"major"` &#124; `"minor"` &#124; `"positive"` &#124; `"alertive"` &#124; `"negative"` | Intent value  | `null`  |
| `children` | `ReactNode`                                                                                          | Subcomponents | -       |

### Intent.useConfig()

```ts
function useConfig():
  | null
  | "major"
  | "minor"
  | "positive"
  | "alertive"
  | "negative";
```

Get configured intent value.

### Intent.Context

A context created by `React.createContext`. It's default value is `null`.
