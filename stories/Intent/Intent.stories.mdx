## API

### Intent

| Property   | Type                                                                             | Description                |
| ---------- | -------------------------------------------------------------------------------- | -------------------------- |
| `value`    | `null` \| `"major"` \| `"minor"` \| `"positive"` \| `"alertive"` \| `"negative"` | **Required** intent value  |
| `children` | `ReactNode`                                                                      | **Required** subcomponents |

### Intent.useConfig

```ts
function useConfig():
  | null
  | "major"
  | "minor"
  | "positive"
  | "alertive"
  | "negative";
```

Get configured intent value.

### Intent.Context

A context created by `React.createContext`. It's default value is `null`.
