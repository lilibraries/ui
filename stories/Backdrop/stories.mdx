## API

| Property               | Type                                                                                        | Description                                                                                                                                                                                                                                                                                            |
| ---------------------- | ------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `children`             | `ReactNode`                                                                                 | Content                                                                                                                                                                                                                                                                                                |
| `blurred`              | `boolean`                                                                                   | Blurred backdrop                                                                                                                                                                                                                                                                                       |
| `transparent`          | `boolean`                                                                                   | Transparent backdrop                                                                                                                                                                                                                                                                                   |
| `animated`             | `boolean`                                                                                   | Perform animation, default is `true`                                                                                                                                                                                                                                                                   |
| `container`            | [`EffectTarget<HTMLElement>`](https://lilibraries.github.io/utils/types#effecttarget-react) | The DOM node where the content should be rendered. It can be `null`, `undefined`, a DOM element, a function returns a DOM element or a ref object refrences a DOM element. Default is `document.body`. This prop can be configured by [`Portal.Config`](?path=/story/others-portal--page#portalconfig) |
| `open`                 | `boolean`                                                                                   | Controlled open                                                                                                                                                                                                                                                                                        |
| `defaultOpen`          | `boolean`                                                                                   | Uncontrolled open                                                                                                                                                                                                                                                                                      |
| `openDelay`            | `number`                                                                                    | Open delay, in milliseconds                                                                                                                                                                                                                                                                            |
| `closeDelay`           | `number`                                                                                    | Close delay, in milliseconds                                                                                                                                                                                                                                                                           |
| `prepared`             | `boolean`                                                                                   | Mount on first time, no matter whether the component is opened                                                                                                                                                                                                                                         |
| `keepAlive`            | `boolean`                                                                                   | Don't unmount content on close                                                                                                                                                                                                                                                                         |
| `closeOnEscape`        | `boolean`                                                                                   | Close when pressing the <kbd>ESC</kbd> key                                                                                                                                                                                                                                                             |
| `closeOnPageHide`      | `boolean`                                                                                   | Close when the page is hidden                                                                                                                                                                                                                                                                          |
| `closeOnWindowBlur`    | `boolean`                                                                                   | Close when the `window` loses focus                                                                                                                                                                                                                                                                    |
| `closeOnBackdropClick` | `boolean`                                                                                   | Close when clicking backdrop                                                                                                                                                                                                                                                                           |
| `onClose`              | `() => void`                                                                                | Close event                                                                                                                                                                                                                                                                                            |
| `onOpened`             | `() => void`                                                                                | Opened event                                                                                                                                                                                                                                                                                           |
| `onClosed`             | `() => void`                                                                                | Closed event                                                                                                                                                                                                                                                                                           |

It accepts all props which the native `<div>` element supports.
