## API

| Property       | Type                                                                   | Description                                                                                                     |
| -------------- | ---------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| `icon`         | `ReactNode`                                                            | Custom icon. It will be hidden when it's not renderable (ie. `null` or `false`).                                |
| `title`        | `ReactNode`                                                            | Confirm title.                                                                                                  |
| `detail`       | `ReactNode`                                                            | Confirm detail.                                                                                                 |
| `content`      | `ReactNode`                                                            | When it is renderable, it will replace the `icon`, `title` and `detail` props.                                  |
| `confirmLabel` | `ReactNode`                                                            | Confirm button label, default is `"Confirm"`.                                                                   |
| `cancelLabel`  | `ReactNode`                                                            | Cancel button label, default is `"Cancel"`. It will be hidden when it's not renderable (ie. `null` or `false`). |
| `confirmProps` | [`ButtonProps`](?path=/story/generics-button--page#api)                | Custom confirm button props.                                                                                    |
| `cancelProps`  | [`ButtonProps`](?path=/story/generics-button--page#api)                | Custom cancel button props.                                                                                     |
| `onConfirm`    | `(event: MouseEvent) => void` \| `(event: MouseEvent) => Promise<any>` | Confirm event.                                                                                                  |
| `onCancel`     | `(event: MouseEvent) => void`                                          | Cancel event.                                                                                                   |

More props are presented in [Popup's document](?path=/story/overlays-popup--page#api), except that the default value of the `arrowed` prop is `true` and the default value of the `placement` prop is `"top"`.
