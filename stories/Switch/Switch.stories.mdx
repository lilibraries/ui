## Props

| Property         | Type                                     | Description                                  |
| ---------------- | ---------------------------------------- | -------------------------------------------- |
| `size`           | `null` &#124; `"small"` &#124; `"large"` | Size                                         |
| `icon`           | `ReactNode`                              | Slider icon                                  |
| `children`       | `ReactNode`                              | Same as `icon` prop, but with lower priority |
| `checkedLabel`   | `ReactNode`                              | Checked label                                |
| `uncheckedLabel` | `ReactNode`                              | Unchecked label                              |
| `loading`        | `boolean`                                | Show spinner on the slider                   |
| `disabled`       | `boolean`                                | Disable change event                         |
| `checked`        | `boolean`                                | Controlled checked value                     |
| `defaultChecked` | `boolean`                                | Uncontrolled checked value                   |
| `onChange`       | `ChangeEventHandler<HTMLInputElement>`   | Change event                                 |
| `inputRef`       | `Ref<HTMLInputElement>`                  | Ref to internal input element                |
| `inputProps`     | `InputHTMLAttributes<HTMLInputElement>`  | Internal input props                         |

It accepts all props which the native `<label>` element supports.
